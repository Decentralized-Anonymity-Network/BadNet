(1) Requirements: 
The prototype system installation requires CentOS system and Python 3.6.

(2) Dependencies:
sudo yum install gcc libevent-devel openssl-devel automake python36 python36-devel -y
sudo ln -s /usr/lib64/libpython3.6m.so.1.0 /usr/lib/libpython3.6.so
pip3 install --upgrade pip
pip3 install web3 eciespy

(3) Setup:
git clone https://github.com/Decentralized-Anonymity-Network/BadNet.git
cd BadNet/2021-08-11
unzip BADNet.zip
cd BADNet
sudo sh badnet.sh (This step will output an Ethereum address)

obtain rETH (Ropsten Ethereum) with generated Ethereum address (Just choose one of these websites):
https://faucet.dimensions.network/ 
https://teth.bitaps.com/

(4) Compile and running:
sudo ./configure
sudo make
cd src/app
sudo ./tor -f /usr/local/BADNet-client/BADNet.client

After waiting for 1 to 2 periods:
sudo cat /usr/local/BADNet-client/lib/MSRI
Start testing after making sure the MSRI has content:
curl --socks5 127.0.0.1:9550 http://checkip.amazonaws.com/
